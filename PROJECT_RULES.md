# プロジェクト・グラウンドルール：歴史人物AI相談所

本プロジェクト（歴史人物AI相談所）の進行において、以下のルールを厳守します。

## 1. コンテクストの完全分離
*   **既存ナレッジの非参照**: グローバルルールに記載されている `Obsidian Vault` 内の思想（DAO論、価値持続スコア、正直者が勝つ物理法則など）は、本プロジェクトにおいては一切参照しません。
*   **独立した意思決定**: 本プロジェクトの要件、デザイン、技術スタックは、プロジェクトディレクトリ（`re` ディレクトリ）内のドキュメントおよびユーザーからの直接的なフィードバックのみに基づいて決定します。
*   **過去文脈の排除**: 過去の別プロジェクトで培われた特定の「勝ちパターン」や「思想」を本プロジェクトに持ち込まないよう、常にゼロベースで思考します。

## 2. コミュニケーション方針
*   **プロジェクト・フォーカス**: 提案や議論は、純粋に「歴史人物AI相談所」をより魅力的なエンターテインメント・アプリにするための視点で行います。
*   **用語の定義**: 本プロジェクト内で使用される用語（例：「親密度」「感情層」）は、本アプリの仕様に準拠した意味でのみ使用します。

## 3. 開発・設計の優先順位
1.  **ユーザー体験（没入感・エンタメ性）**の最大化
2.  **歴史人物のペルソナ再現度**の向上
3.  **現代とのギャップが生む「面白さ」**の追求

本ルールはプロジェクトの全工程において、他の如何なる一般的な慣習や過去のルールよりも優先されます。
